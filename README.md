# food-reels-website
A food reel-based website where users can watch food videos and place orders for items they like.

## Features
- Scrollable food reels interface
- Food item details
- Order option for selected dishes
- Simple and user-friendly design

- ## Technologies Used
- MongoDB
- Express.js
- React.js
- Node.js
- HTML
- CSS
- JavaScript

## Purpose
This project was built to practice full-stack web development using the MERN stack and to gain hands-on experience in both front-end and back-end technologies.
